# Maquetado estilo WhatsApp

Proyecto base, comisión 999200928.  
Trabajo práctico realizado UTN.

## Descripción
Maquetado de una interfaz web inspirada en el diseño de WhatsApp, desarrollado exclusivamente con HTML y CSS.

## Tecnologías utilizadas
- HTML
- CSS
- Git / GitHub

## Estructura del proyecto
- index.html (estructura principal del maquetado)
- style/
  - style.css (estilos del proyecto)

Los recursos SVG se encuentran integrados directamente dentro del archivo HTML.

## Cómo ejecutar el proyecto
1. Descargar o clonar el repositorio.
2. Abrir el archivo `index.html` en cualquier navegador web.
